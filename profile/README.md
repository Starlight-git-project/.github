## O que é a Starlight?

Seja bem-vindo! A **Starlight Git Project** é uma org com projetos educacionais para analistas e engenheiras de dados que precisam aprender Git e GitHub com profundidade, não só os comandos básicos, mas as três camadas que separam quem aprende e pratica no dia a dia.

> Credenciais vazadas no GitHub. Dados de produção apagados.  
> Isso acontece. E acontece com quem não aprendeu as três camadas.

---

## As três camadas

| Camada | O que você aprende |
|--------|-------------------|
| ⭐ **1 - Boas práticas individuais** | Commits semânticos, `.gitignore` correto, variáveis de ambiente, estrutura de projeto |
| ⭐⭐ **2 - Governança de pipeline** | Branches, pull requests, code review, pre-commit hooks, secret scanning |
| ⭐⭐⭐ **3 - Infraestrutura organizacional** | GitHub Orgs, permissões, políticas de repositório, least privilege, audit log |

## Para quem é isso?

- Analistas de dados que usam Git mas nunca aprenderam direito
- Engenheiras de dados em início de carreira
- Profissionais que vieram do Excel/BI e estão migrando para código
- Qualquer pessoa de dados que já teve medo de dar `git push` errado
